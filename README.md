Trabalho realizado em turma no curso presencial Planejamento Estatístico e Análise de Dados da UNESP araraquara, que participei como aluno especial.
Além de toda a teoria básica sobre estatística, a parte prática envolveu a limpeza e tratamento dos dados, análise exploratória e um pouco de inferência estatística.

Os softwares mais utilizados foram o Excel, JAMOVI e JASP, mas também usei SQL e POWER BI para treinar um pouco de banco de dados e visualização com ferramenta de visualização mais conhecida no mercado e não tão direcionada para fins acadêmicos.
